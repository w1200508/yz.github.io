# 如何自己修改 / 添加内容(无需懂代码)

你的网站只有这几个文件,**平时你只需要动 `content.js` 这一个文件**:

| 文件 | 作用 | 你要不要动它 |
|---|---|---|
| `content.js` | 所有文字、项目、图片清单 | ✅ 主要改这个 |
| `assets/` 文件夹 | 存放所有图片 | ✅ 上传新图到这里 |
| `index.html` | 首页(自动读取 content.js) | ❌ 一般不用动 |
| `project.html` | 项目详情页模板(自动生成) | ❌ 一般不用动 |

---

## 方式一:在浏览器里直接改(推荐,最简单)

1. 打开你的仓库:https://github.com/w1200508/yz.github.io
2. 点开 `content.js` 文件 → 点右上角的 **铅笔图标 ✏️**(Edit this file)
3. 改文字:把引号 `"..."` 里的内容换成你的
4. 改完点右上角绿色 **Commit changes** 按钮 → 再点一次确认
5. 等 1~2 分钟,刷新你的网站,内容就更新了

## 常见操作

### ✏️ 改一个项目的标题 / 介绍
在 `content.js` 找到那个项目,改 `title`(标题)、`subtitle`(副标题)、`body`(正文段落)后面引号里的字。

### 🖼️ 换 / 加一张图片
1. 先上传图片:在仓库里进入 `assets` 文件夹 → 点 **Add file → Upload files** → 把图片拖进去 → Commit。
2. 再到 `content.js` 里引用它:
   - 换封面:把 `cover: "assets/旧图.jpg"` 改成你的新文件名
   - 加到图集:在 `gallery: [ ... ]` 的方括号里加一行,例如
     ```
     gallery: ["assets/proj-brick.jpg", "assets/新图.jpg"],
     ```
   - ⚠️ 文件名要和上传的完全一致(包括大小写和 .jpg/.png 后缀),建议用英文名、不要有空格。

### ➕ 添加一个全新项目
1. 在 `content.js` 里复制任意一整段 `{ ... },`(从 `{` 到 `},`)
2. 粘贴到最后一个项目的 `},` 后面
3. 把里面的内容改成新项目;`id` 必须改成一个**新的、唯一的英文单词**(不能有空格),例如 `"newproject"`
4. Commit 后首页会自动多出一张卡片,点进去就是它的详情页

### 🗑️ 删除一个项目
把那个项目从 `{` 到 `},` 整段删掉即可。

## 小提示
- 引号 `"` 必须成对出现;如果正文里要用引号,请用中文引号 `“ ”` 或单引号 `'`。
- 每个项目块之间用逗号 `,` 隔开,别漏掉。
- 改错了不用怕:GitHub 会保留每次修改历史,可以随时回退。
- 想让加载更快(图片压缩),随时找我帮忙。
