# 如何自己修改 / 添加内容(无需懂代码)

你的网站支持**中英文切换**。平时你**只需要动 `content.js` 这一个文件**。

| 文件 | 作用 | 你要不要动它 |
|---|---|---|
| `content.js` | 所有文字(中英双语)、项目、图片清单 | ✅ 主要改这个 |
| `assets/` 文件夹 | 存放所有图片 | ✅ 上传新图到这里 |
| `index.html` / `project.html` | 首页与详情页(自动读取 content.js) | ❌ 一般不用动 |

---

## 双语怎么写
文件里每一段文字都长这样,英文和中文各一句:
```
title:{ en:"Inflatable Pajama", zh:"充气睡衣" },
```
- 改英文 → 改 `en:` 后面引号里的字
- 改中文 → 改 `zh:` 后面引号里的字
- ⚠️ 中文里如果要加引号强调,请用中文引号 `“ ”`,**不要用英文的 `"`**(会把这行弄坏)。

---

## 在浏览器里直接改(推荐,最简单)
1. 打开仓库:https://github.com/w1200508/yz.github.io
2. 点开 `content.js` → 点右上角 **铅笔图标 ✏️**
3. 改引号里的内容
4. 点绿色 **Commit changes** → 再确认一次
5. 等 1~2 分钟,刷新网站即可看到更新

## 常见操作

### ✏️ 改项目标题 / 介绍
在 `PROJECTS` 里找到那个项目,改 `title` / `subtitle` / `body` 的 en、zh。
`body` 是正文段落,每段是一个 `{en:"...",zh:"..."}`,可以增减。

### 🖼️ 换 / 加图片
1. 进 `assets` 文件夹 → **Add file → Upload files** → 拖入图片 → Commit。
2. 回到 `content.js`:
   - 换封面:改 `cover:"assets/你的新图.jpg"`
   - 加进详情页图集:在 `gallery:[ ... ]` 里加一行 `"assets/你的新图.jpg"`(逗号隔开)
   - 文件名要和上传的完全一致(建议英文、无空格)。

### ➕ 加新项目
1. 复制 `PROJECTS` 里任意一整个 `{ ... },` 块
2. 粘到列表里,改内容
3. `id` 改成一个**新的唯一英文词**(无空格),例如 `"newproject"`
4. `wide:true` = 横向大卡片,`wide:false` = 普通卡片
5. Commit 后首页自动多一张卡片,点进去就是详情页

### 🗑️ 删项目
把那一整个 `{ ... },` 块删掉。

### 改"关于我"(简历)
在 `RESUME` 里改 `summary`(简介)、`experience`(经历)、`education`(教育)、`publications`(发表)、`skills`(技能)、`awards`(奖项)。

## 小提示
- 引号必须成对;每个 `{ ... }` 之间用逗号 `,` 隔开。
- 改错了不怕:GitHub 保留每次历史,可随时回退。
- 想让图片加载更快(压缩),随时找我帮忙。
